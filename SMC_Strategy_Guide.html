Strategy Components
🧠

Order Blocks (OB)

Last opposing candle before impulsive move. Bullish OB = demand zone, Bearish OB = supply zone.

⚖️

Fair Value Gaps (FVG)

3-candle imbalance where price moved too fast. Price tends to return and fill these gaps.

🔄

BOS / CHoCH

Break of Structure signals trend continuation. Change of Character signals early reversal.

💧

Liquidity Sweeps

SSL sweep = stop hunt below lows → bullish reversal. BSL sweep = above highs → bearish reversal.

📊

Volume Surge

Volume 1.5× above MA confirms institutional participation. Volume delta filters directional flow.

📈

VWAP + EMA

VWAP as fair value anchor. EMA 20/50/200 stack defines macro trend on each timeframe.
