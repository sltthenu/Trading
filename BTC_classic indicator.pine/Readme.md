📋 Day Trading Workflow (Step by Step)
Before the session:

Open BTC/USDT on 4H — check if trend background is 🟢 green (bullish) or 🔴 red (bearish)
Note the 4H EMA 200 level — that's your key support/resistance for the day

During the session:

Switch to 1H chart with the indicator running
When a signal fires, cross-check — does it match the 4H trend direction?
If yes → enter on the next candle open after signal bar closes
If the signal goes against the 4H trend → skip it

Exit rules:

Hit TP line → close full position
Hit SL line → close and wait for next setup
If 4H trend flips direction mid-trade → consider early exit
