Every professional trader has a documented loss rate of 30-45%.

Your indicator with 60-65% WR at score 5+/8 is genuinely good.

SECTION 5 — POSITION SIZING & RISK MATH
================================================================================

5.1  POSITION SIZE FORMULA (step by step)
-------------------------------------------
  1. Risk Amount  = Account Balance x 0.01  (1% risk)
  2. Risk in $    = Entry Price - SL Price
  3. Position     = Risk Amount / Risk in $

  EXAMPLE:
  Account = $10,000 | Risk = 1% = $100
  Entry = $68,000   | SL = $65,600  | Risk per BTC = $2,400
  Position = $100 / $2,400 = 0.0417 BTC

5.3  KELLY CRITERION (at 62% WR + 2.5R)

  Formula : f* = (p x b - q) / b

SECTION 8 — DASHBOARD READING GUIDE

The top-right table on your chart — what each row means:

  Daily Bias      : The macro direction (Daily timeframe)
                    GREEN BULL = macro uptrend
                    RED BEAR   = macro downtrend
                    YELLOW NEUT = transitioning

  4H Trend        : Your primary trade direction filter
                    Must match your intended direction for full score

  1H Momentum     : Confirms short-term direction
                    MACD + EMA alignment on 1H

  BUY Alignment   : 4H ALIGNED = 4H supports buying
                    TRIPLE VETO = Daily+4H+1H all bearish, no buy

  SELL Alignment  : 4H ALIGNED = 4H supports selling
                    TRIPLE VETO = Daily+4H+1H all bullish, no sell

  RSI             : Current RSI reading
                    BLOCKED BUY  = RSI below 30 (too oversold)
                    BLOCKED SELL = RSI above 70 (too overbought)
                    OK = in tradeable range

  Vol Spike       : YES = institutional volume present
                    Required for highest-conviction entries

  Liq Sweep UP    : YES = stop hunt below recent lows just happened
                    Precedes the highest WR buy setups

  Liq Sweep DOWN  : YES = stop hunt above recent highs just happened
                    Precedes the highest WR sell setups

  Buy Score       : X/8 — how many confluence layers active for buys
  Sell Score      : X/8 — how many confluence layers active for sells

  Stretch %EMA50  : How far price is from EMA50
                    BLOCKED OB  = stretched too far above (>4%)
                    BLOCKED OS  = stretched too far below (<-4%)

  Cooldown        : READY = can take new signal
                    X bars left = locked, wait

  Signal          : BUY READY / SELL READY / WAITING
                    This is your final go/no-go indicator
